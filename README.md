# drivio-backend 

USER AUTHENTICATION
├─ User registration (Customer & Admin)
├─ Secure login
└─ Password hashing

CAR MANAGEMENT
├─ Add cars to database (Admin only)
├─ Display all cars (API endpoint)
├─ Search/filter cars (by brand, type, price)
└─ Update car status (available, rented, maintenance)

BOOKING SYSTEM
├─ Create booking (check availability)
├─ View bookings (customer & admin)
├─ Update booking status
└─ Cancel booking

PAYMENT PROCESSING
├─ Payment integration (Stripe/PayPal)
├─ Store payment records
└─ Generate payment receipts

REPORTS (MIS)
├─ Booking report (total rentals by timeframe)
├─ Revenue report (earnings breakdown)
├─ Customer report (rental history)
└─ Car utilization report (most/least rented)
